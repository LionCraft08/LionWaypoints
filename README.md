# LionWaypoints 
This plugin provides Server side waypoints. It works without LionAPI, but has more features with it.

# Features
- Quick-Waypoints
    use the command /pos <name> to create a waypoint at your position and to get the coordinates of it in chat.
- Waypoints 
    Waypoints can be managed using the /wp command. You can open a management UI, create and delete Waypoints at specific locations and more.
- Ui
    You can manage Waypoints in the UI. this allows you to change the icon, dimension and location of a waypoint, and track it. See the next point for more
- Tracking 
    You can open a compass that points to your location.
- - Vanilla: Bind a compass to your location
- - Also kind of vanilla: Use the boss bar to display the direction and distance to the location
- - Resource Pack: Use the resource pack to upgrade the visuals of the boss bar
- - Modded: Use LionDisplays to display a compass on your screen. This is better for server performance because it is calculated client side.
